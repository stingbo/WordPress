####个人博客系统
* css（文件夹）：存放 css 文件
* genericons（文件夹）：存放字体文件
* inc（文件夹）：存放 php 文件，文件内一般是自定义函数
* js（文件夹）：存放 js 文件
* languages（文件夹）：存放语言文件
* template-parts（文件夹）：为了代码简洁，作者把一些代码（可能重复用到的）分开放到了此文件夹下
* 404.php：出现404错误时使用的文件
* archive.php：分类页面
* comments.php：评论
* footer.php：页脚
* functions.php：定义函数
* header.php：页头
* image.php：图片
* index.php：首页
* page.php：页面（page）
* rtl.css：关于字体的一些样式
* screenshot.php：主题缩略图，在后台显示
* search.php：搜索页面
* searchform.php：搜索框
* sidebar.php：侧边栏
* sidebar-content-bottom.php：侧边栏底部
* single.php：文章（post）页面
* style.css：样式文件
